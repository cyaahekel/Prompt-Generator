# Installation

## 1. Clone Repository

```bash
git clone https://github.com/cyaahekel/Prompt-Generator
```

## 2. Open Project Folder

```bash
cd Prompt-Generator
```

## 3. Install Dependencies

```bash
npm install
```

## 4. Setup Environment

Create file:

```env
.env.local
```

Then add your Gemini API key:

```env
GEMINI_API_KEY=your_gemini_api_key
```

---

# Run Development Server

```bash
npm run dev
```

Open in browser:

```bash
http://localhost:5173
```

---

# Build Production

```bash
npm run build
```
